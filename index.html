Scientific Calculator | Sizelenkosi

Tech Used:

  • HTML

    The HTML structure of the calculator included all essential components like buttons for numbers and operators, an input field to display calculations, and a simple layout to guide the user. Semantic HTML ensured accessibility, making the calculator usable with screen readers.

  • CSS

    It included hover effects on buttons to improve user interaction, and flexbox was used to align items neatly within the container. Media queries made the calculator responsive, ensuring usability on smaller screens.

  • Javascript

    JavaScript was the core of the calculator’s functionality, handling all arithmetic operations and updating the display in real time as the user interacted with the buttons. Functions were created for addition, subtraction, multiplication, division, and clearing/resetting the calculator.
